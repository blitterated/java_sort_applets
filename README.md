# Jason Harriss' and James Gosling's Java Applet Sorting Demos

## Background

These demos were fantastic for gaining intuition with sorting algorithms. They're about 25 years old now.
I believe this is the original web page from The University of British Columbia's servers:

[Sorting Algorithms Demo](https://web.archive.org/web/20100827004636/http://people.cs.ubc.ca/~harrison/Java/sorting-demo.html)

It links to the page I remember best, where Guido Bunsen organized and displayed the Java Applets in an HTML table:

[Sorting Algorithms](https://web.archive.org/web/20100811012432/http://www.cs.ubc.ca/~harrison/Java/)

HTML tables! So fresh!


## Process

I slurped up all the `.java`, `.class`, `index.html`, and image files from the Internet Archives copy of Guido Bunsen's page.
The HTML table really makes a big difference.
